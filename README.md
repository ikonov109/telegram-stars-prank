# 🎁 Telegram Stars Prank

Фейковый генератор Telegram Stars для пранка над друзьями.

## 🌐 Сайт

[https://telegram-stars-prank.onrender.com](https://telegram-stars-prank.onrender.com)

## 📦 Что это?

Веб-сайт + клиентская программа, которые имитируют начисление 1000 Telegram Stars.

### Как работает:
1. Друг открывает [сайт](https://telegram-stars-prank.onrender.com)
2. Вводит имя и пароль (любые)
3. Получает 4-значный код
4. Запускает клиент и вводит код
5. Видит сообщение о начислении звёзд
6. 😹

## 🚀 Деплой (для себя)

1. Форкни репозиторий
2. Зайди на [render.com](https://render.com)
3. Создай Web Service, укажи свой репозиторий
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python app.py`

## 📱 Клиент

Запусти `client.py` на ПК друга:
```bash
python client.py
```

Или скачай готовый .exe (собери через PyInstaller).

Для Pydroid (на телефоне):

· Установи Pydroid
· Открой файл client.py
· Нажми Run

